Web Dev Team Project Setup:
1) Download Python and Flask
2) Navigate to the "Flask_Webpage" directory
3) Run flask with either command "python flaskpage.py" or "flask run" with environment variables set

* Windows Specific Commands
- Set environment variable on windows: $env:Flask_APP = "flaskpage.py"
- Debug Mode on Windows: $env:FLASK_DEBUG = 1
